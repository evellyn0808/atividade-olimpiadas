README – Footer Responsivo com HTML e CSS

Este projeto consiste na criação de um rodapé (footer) moderno utilizando HTML5 e CSS3, inspirado em layouts profissionais de sites de turismo e blogs.

O footer é dividido em três partes principais:

Uma faixa superior com logos
Uma área principal com quatro colunas de conteúdo
Uma parte inferior com links e informações de copyright

Tecnologias utilizadas:

HTML5
CSS3
Flexbox

Estrutura do projeto:

index.html → estrutura da página
style.css → estilização
pasta de imagens → logos e outros elementos visuais

Funcionalidades:

Layout organizado e moderno
Alinhamento com Flexbox
Separação em seções bem definidas
Área de contato com telefone, email e endereço
Seção de posts recentes
Campo para inserir email (newsletter)
Botão estilizado para inscrição

Estrutura do layout:

Top Footer:
Área com fundo azul onde ficam os logos alinhados horizontalmente.

Main Footer:
Dividido em quatro colunas:

Texto institucional
Informações de contato
Publicações recentes
Campo de inscrição (newsletter)

Bottom Footer:
Parte final com:

Links como Privacy Policy, Termos e FAQ
Informação de copyright
